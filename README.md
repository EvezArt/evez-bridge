# EVEZ Intelligence Bridge — Dashboard + Audit API

Real-time dashboard and audit interface for the EVEZ ecosystem.

## What It Does
- Live service health dashboard
- MAES event stream visualization
- CAIN audit trail browser
- System metrics and performance graphs

## API

```bash
curl http://localhost:8083/health
curl http://localhost:8083/status
curl http://localhost:8083/audit
```
